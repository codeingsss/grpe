# GRPE

GRPE는 Gemini를 사용하여 자연어 지시를 실행 가능한 셸 명령으로 변환하는 Python 기반 AI 명령줄 도구입니다.

## 프로젝트 구조

- SDK
  - 셸 버전에서 사용하는 핵심 Python SDK가 포함되어 있습니다.
- v1 shell
  - 로컬에서 실행되는 인터랙티브 셸 버전입니다.
- v2 shell
  - 설치 스크립트와 명령 등록 기능을 포함한 더 완성도 높은 셸 버전입니다.

## 필요 조건

- Python 3.8 이상
- pyfiglet
- google-genai

필수 패키지는 다음 명령으로 설치할 수 있습니다.

```bash
pip install pyfiglet google-genai
```

## 사용 개요

1. 사용할 버전을 선택합니다.
   - v1 shell: 간단한 인터랙티브 경험을 원할 때
   - v2 shell: 더 완성도 높은 설정 과정을 원할 때
2. 해당 폴더의 README에 안내된 절차를 따릅니다.
3. 프롬프트에 Gemini API 키를 입력합니다.
4. 자연어로 명령을 입력하기 시작합니다.

## 참고 사항

- 이 프로젝트는 Gemini를 사용해 프롬프트를 셸 명령으로 변환합니다.
- 명령은 스크립트가 실행되는 환경에서 실행됩니다.
- SDK와 셸 버전은 실용적인 명령 생성 및 로컬 자동화 작업에 사용됩니다.
